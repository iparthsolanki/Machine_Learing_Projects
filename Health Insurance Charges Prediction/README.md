---

## Overview

Health Insurance Charges Prediction is an end-to-end Machine Learning project that estimates insurance premiums using customer demographics, medical history, and lifestyle information.

The project includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression Model
- Model Evaluation
- Streamlit Web Application
- Real-time Insurance Charge Prediction
